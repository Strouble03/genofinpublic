- **App won’t start:**  
  Make sure you installed the software by running the installer as administrator.

- **Cannot connect to exchange:**  
  Double-check your API keys. Ensure your keys are active and have the right permissions (trading enabled).

- **Trades not executing:**  
  Confirm your account has enough balance. Check if the chosen strategy is active and running.

- **Software crashes:**  
  Close the app and restart your computer. Try running the app again. If it fails, check for Windows updates or contact support.
